This application is an ongoing project whih is going to visualize the git log results with different graphs.

      
![](http://docutils.sourceforge.net/docs/user/rst/images/ball1.gif) **Requirements**
                            					          

 - PyQtGraph 0.9.9+ 
 - Python 2.7 and 3+
 - PyQt or PySide 4.8+
 - NumPy
 - python-opengl
 - mongodb 2.4.9+
